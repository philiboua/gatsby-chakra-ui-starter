# Billboard

`Billboard` component is used at the top of the page to provide main information about the page.

## ** Preview **

### Billboard

<Canvas>
  <Story name="Billboard">
    <Billboard headline="Get started with Chakra UI" caption="CHAKRA UI" />
  </Story>
</Canvas>

The billboard component is flexible and can multiple slices. You can add the following slices : `body`, `media`, `footer`. Each slices will be added at specific location inside the component.

<Canvas>
  <Story name="Billboard with custom slices">
    <Billboard headline="Get started with Chakra UI" caption="CHAKRA UI">
      <Slice __TYPE="body">
        <Text type="subtitle.medium"> This is the body slice</Text>
      </Slice>
      <Slice __TYPE="media">
        <GridItem>
          <Text type="subtitle.medium">
            This is the media slice, it's a great place to add images, or videos
          </Text>
        </GridItem>
      </Slice>
      <Slice __TYPE="footer">
        <Text type="subtitle.medium">
          This is the footer slice, we can add a gallery of logos for example,
          the world is yours :)
        </Text>
      </Slice>
    </Billboard>
  </Story>
</Canvas>
