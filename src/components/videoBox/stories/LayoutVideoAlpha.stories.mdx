# VideoBox - VideoLayoutAlpha

`VideoLayoutAlpha` allow any user to pause, play, mute and unmute the video. The video controls are located at the bottom of the video.

## How to use it

1. Import the `VideoBox` and `VideoLayoutAlpha` Components

```javascript
import { VideoBox, LayoutVideoAlpha } from "@src/components"
```

2. Set up the component with the desired props

```javascript
<VideoBox
  loop
  videoMp4SrcURL={video}
  height="500px"
  width="100%"
  borderRadius="sm"
>
  <LayoutVideoAlpha />
</VideoBox>
```
